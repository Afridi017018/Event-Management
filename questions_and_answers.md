###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: ReferenceError: greetign is not defined

<i>The code contains a typo in the variable name "greetign" instead of "greeting," leading to a ReferenceError since "greetign" is not defined. The correct code should assign an empty object to "greeting" and log "{}" to the console.</i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: NaN

<i>When adding a number (1) and a string ("2"), JavaScript attempts to coerce types, resulting in concatenation rather than addition, leading to the string "12," which is then converted to NaN (Not-a-Number).</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: Hi there, undefined

<i>The sayHi function expects an argument name, but when it's called without any argument (sayHi()), name becomes undefined. Therefore, the function returns "Hi there, undefined."</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: 3

<i>The forEach loop iterates through each element in the nums array. It checks if each element (num) is truthy (non-zero in this case) and increments the count variable accordingly. Since three elements in the array (1, 2, and 3) are truthy, the final value of count is 3.</i>

</p>
</details>
